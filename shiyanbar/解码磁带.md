# 解码磁带

[解码磁带](http://www.shiyanbar.com/ctf/1891)

```python
message = """o____o_
oo__o_o
oo_o__o
oo_o_o_
oo_o__o
oo_ooo_
oo__ooo
_o_ooo_
"""


def decode(message):
    return "".join(chr(int(i.replace("o", "1").replace("_", "0"), base=2)) for i in message.split())


result = decode(message)
print(result)
# Beijing.
```
